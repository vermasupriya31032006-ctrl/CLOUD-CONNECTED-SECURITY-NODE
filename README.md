# CLOUD-CONNECTED-SECURITY-NODE


## Overview

Cloud Connected Security Node is an IoT project based on the ESP32 microcontroller. The system measures distance using an HC-SR04 ultrasonic sensor and publishes real-time telemetry data to the cloud using the MQTT protocol. The project demonstrates cloud connectivity, wireless communication, and IoT monitoring.

## Features

- ESP32-based IoT device
- HC-SR04 Ultrasonic Sensor
- Real-time distance measurement
- Wi-Fi connectivity
- MQTT communication
- HiveMQ Cloud integration
- Live cloud telemetry

## Hardware
